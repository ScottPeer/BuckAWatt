Don't waste electricity, it costs a Buck A Watt!


The "Buck A Watt" rule:

For a typical electricity user, using one watt continuously for one year costs about one dollar.


The "Penny An Hour" rule:

It costs about a penny an hour for the electricity to run medium-power devices like an incandescent light bulb, four CFL bulbs, typical stereo, small TV, fan, or laptop computer.
